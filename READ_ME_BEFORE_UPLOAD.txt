SAIL Research Solutions - FINAL GitHub Upload Package

IMPORTANT:
Do not upload this ZIP file directly to GitHub Pages.
First unzip this package. Then upload the extracted files and folders to your GitHub repository.

Files/folders that must be visible in the GitHub repository root:
- index.html
- about.html
- services.html
- publication-support.html
- submit-requirement.html
- contact.html
- faq.html
- policies.html
- assets/

Upload steps:
1. Unzip this package.
2. Open the extracted folder named SAIL_Research_Solutions_FINAL_GitHub_Upload.
3. In GitHub repository, click Add file > Upload files.
4. Drag all files and the assets folder.
5. Commit changes with message:
   Final upload with mobile menu fix
6. Wait 5 minutes.
7. Test mobile using:
   https://www.sailresearchsolutions.com/?v=mobilefix2

If GitHub does not allow dragging the assets folder:
1. Upload all HTML files first.
2. Open the existing assets folder in GitHub.
3. Upload only style.css and script.js from the assets folder.
